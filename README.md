# 📅 Hệ thống Lịch Cá Nhân (Personal Calendar)

Chào mừng bạn đến với dự án **Personal Calendar **! Đây là một hệ thống quản lý lịch cá nhân chuyên nghiệp, được thiết kế để giúp bạn tối ưu hóa thời gian và năng suất hàng ngày.

### 🌟 Điểm nổi bật:
- **Quản lý sự kiện & công việc:** Giao diện trực quan, hỗ trợ kéo thả và nhắc nhở thông minh.
- **Phân tích năng suất:** Biểu đồ trực quan giúp bạn theo dõi thói quen và tiến độ công việc.

### 💻 Công nghệ sử dụng:
- **Frontend:** ReactJS.
- **Backend:** Node.js, ExpressJS.
- **Database:** MySQL.
